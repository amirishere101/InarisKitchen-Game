# Screenshots

Place your game screenshots in this folder.

## Recommended Files

- `banner.png` - Main banner image (recommended: 1920x1080 or 800x400)
- `gameplay-01.png` through `gameplay-04.png` - Gameplay screenshots
- `logo.png` - Game logo

## Tips

- Use PNG format for best quality
- Recommended resolution: 1920x1080 or 1280x720
- Keep file sizes reasonable (under 2MB each for web viewing)
- Capture interesting moments: cooking, dialogue, characters, menus
