# Screenshots

Place your game screenshots in this folder.

## Required Files

| Filename              | Description                                           |
| --------------------- | ----------------------------------------------------- |
| `banner.png`          | Main banner image (recommended: 1920x1080 or 800x400) |
| `plant-growth-01.png` | Plant growth screenshot 1                             |
| `plant-growth-02.png` | Plant growth screenshot 2                             |
| `day-night-01.png`    | Daytime screenshot                                    |
| `day-night-02.png`    | Nighttime screenshot                                  |
| `fishing-01.png`      | Fishing screenshot 1                                  |
| `fishing-02.png`      | Fishing screenshot 2                                  |
| `customization.png`   | Restaurant customization screenshot                   |

## Tips

- Use PNG format for best quality
- Recommended resolution: 1920x1080 or 1280x720
- Keep file sizes reasonable (under 2MB each for web viewing)
- Capture interesting moments that showcase each feature
